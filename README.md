## 项目简介

该项目是基于Vue全家桶的pc商城，该项目是自己在vue学习的历程中练手和学习的项目

## 项目如何运行

``` bash
# Install dependencies
npm install

# Serve with hot reload at localhost:8080
npm run dev

# Build for production with minification
npm run build
```
